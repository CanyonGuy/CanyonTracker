# CanyonTracker
Track canyon descents, GPS routes, rappel stats, photos, and notes. Import or export your canyon history anytime.

## App-Support / Bug Reporting
## ⚠️ How to Request Support

Please read these rules before opening a public ticket:
1. **Never post private data:** Do not share your real name, email, phone number, billing receipts, or account passwords.
2. **Search first:** Look through the open issues list to see if someone else already reported your bug.
3. **Use the template:** Fill out the automated form completely so we can reproduce your issue.
